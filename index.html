Revenge of the Octocat

This is a top-down shooter written in Java 7 between December 2014 and January 2015 for my AP Computer Science class. I worked with jelliott37 for this project, though I unintentionally sped ahead of him while he was on vacation.

Downloading

$ cd some/directory
$ git clone http://www.github.com/elc1798/revenge-of-the-octocat
$ cd revenge-of-the-octocat

If you don't have 'git' installed, you can use wget:

$ cd some/repository
$ wget https://github.com/elc1798/revenge-of-the-octocat/archive/master.zip
$ unzip master.zip
$ rm master.zip #Optional

Running

Compile the code, the run:

$ javac Driver.java #Or javac Game.java
$ java Driver       #Or java Game
